# A50 六维度评分规则 v2.2

## 评分维度总览

| 维度 | 权重 | 满分 | 计算方式 |
|------|------|------|----------|
| 护城河 | 25% | 25分 | 5子项各5分 |
| 成长性 | 20% | 20分 | 历史8+趋势6+空间4+景气2 |
| 盈利质量 | 20% | 20分 | ROE+毛利率+现金流+分红 |
| 估值安全边际 | 25% | 25分 | PE/PB分位+股息率 |
| 催化剂 | 10% | 10分 | 催化已发生9-10分 |
| 风险扣分 | - | - | 治理+财务+行业+竞争 |

## 详细评分标准

### 1. 护城河（25分 = 5×5）

#### 品牌 (0-5)
| 评分 | 关键词 |
|------|--------|
| 5 | 品牌=硬通货、绝对龙头、全球第一、垄断地位、绝密配方 |
| 4 | 龙头、第一、领先、核心供应商 |
| 3 | 前三、主要、优势、知名 |
| 2 | 一般 |

#### 成本 (0-5)
| 评分 | 关键词/条件 |
|------|-------------|
| 5 | 成本领先、极致成本 |
| 4 | 全球第一/最大、市占率第一、绝对龙头 |
| 3 | 成本优势、规模效应、一体化 |
| 2 | 一般 |
| 额外 | 毛利率>70%→+1, >50%→+1 |

| 评分 | 关键词 |
|------|--------|

| 3 | 技术壁垒、技术领先、专利、研发投入 |

| 评分 | 关键词 |
|------|--------|

| 4 | 资源优势、特许经营 |

#### 转换成本 (0-5)
| 评分 | 关键词 |

| 4 | 高转换成本、客户粘性高 |

## 评级标准

| 总分 | 评级 | 含义 |
|------|------|------|
| 90-100 | S | 极度低估/确定性极高 |
| 85-89 | A+ | 明显低估/逻辑清晰 |
| 80-84 | A | 低估/逻辑清晰 |
| 75-79 | B+ | 合理偏低/逻辑通顺 |
| 70-74 | B | 合理/逻辑通顺 |
| 60-69 | C | 估值偏高/需要观察 |
| <60 | D | 高风险/逻辑有硬伤 |

## 评级上限约束

- 有重大风险事件 → 上限B+
- 关键数据缺失 → 上限B
- 行业处于下行周期 → 上限B+

## 数据缺失处理

| 缺失项 | 影响 |
|--------|------|
| 估值（PE/PB都无） | 估值维度≤70%（≤17分） |
| ROE | 盈利质量维度≤70%（≤14分） |
| 营收/净利增速 | 成长性维度≤70%（≤14分） |
| 股息率 | 估值安全边际影响 |

## 金融行业替代口径

银行：PE用PB替代，ROE用ROAE替代
保险：PE用P/EV替代，NBV增速作为成长性指标
券商：PB作为主要估值指标
